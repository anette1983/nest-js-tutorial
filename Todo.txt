# ИНСТРУКЦИЯ
* по поводу базы данных и .env переменных будут уроки. если вы скачали этот архив сразу как только вышел ролик про обзор проекта то скорее всего этих роликов еще нет. Вам нужно либо разобраться самому либо дождаться когда выйдут эти ролики. Они уже готовы и это не займет много времени.

[] - Убедитесь, что у вас установлен nodejs ( https://nodejs.org/en )
[] - Установите базу данных PostgreSQL и создайте саму базу ( https://www.postgresql.org )
[] - Перейдите в папку CLIENT и установите там все зависимости командой npm install
[] - Перейдите в папку SERVER и установите там все зависимости командой npm install
[] - Внутри папки SERVER создайте файл с именем .env (обязательно с точкой) и добавьте туда необходимые значения в переменные:
    - JWT_SECRET=любая рандомная строка 
    - DB_HOST=тут хост postgresql который вы указывали при регистрации. по умолчанию localhost
    - DB_PORT=тут порт postgresql который вы указывали при регистрации. по умолчанию 5433
    - DB_USERNAME=тут имя пользователя postgresql который вы указывали при регистрации. по умолчанию вроде бы postgres
    - DB_PASSWORD=тут пароль postgresql который вы указывали при регистрации. 
    - DB_NAME=тут название таблицы которую вы создали под проект postgresql

[] - Для запуска сервера используйте команду npm run start:dev
[] - Для запуска клиента используйте команду npm run dev
